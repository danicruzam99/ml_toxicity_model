# ML toxicity model with RF
Toxicity prediction with Random Forest Model.

This project predicts the toxicity of molecules based on their drug_likeness, specifically LogP(membrane permeability), molecular weight (MW) and TPSA.

BIOFACQUIM, a database containing natural products from LATAM, was used.

## Tools
-Python
-RDKit
-scikit-learn

Model prioritizes detection of toxic compounds to reduce false negatives.
